-------------------------------- Command Line ---------------------------------

Questa linea di comando potrebbe introdurre un sistema più rapido per lanciare
comandi complessi.

Ecco alcune idee da valutare :

- Implementazione comandi per il settaggio di configurazioni varie
- Consentire di specificare comandi per il lancio dei tools di F12
- Implementazione comandi per l'apertura e il salvataggio dei file
- Implementazione comandi per lo spostamento della posizioen del cursore nel testo
- Implementazione comandi di ricerca e/o sostituzione
- Implementazione comandi di gestione clipboard multipla
