------ Funzione di Ricerca definizione Classi/Funzioni/Variabili -------

Questa funzionalità, dedicata a tutti i tipi di file gestiti da CTAGS consente, una volta identificata una parola di cercare all'interno di un'intero progetto alla ricerca della sua definizione, sia essa una funzione, una variabile, una costante ecc...

Come fare ?

Dato il file corrente, il software ne identifica l'estensione, con la quale viene ricavato il linguaggio (funzione getNomeLinguaggioCorrente), poi :

- capito il linguaggio si determina l'elenco estensioni ricavandolo da "file.patterns.xxx"
- si determina la posizione del file di progetto (vedi RGestPrj). File che identifica la cartella principale dalla quale iniziare ad effetuare la ricerca
- si e si esegue compone il file bat che lancia CTAGS
- nel pannello di output si mostrano i risultati

Nel caso in cui non venga trovato il file di progetto, la ricerca inizierà dalla cartella del file corrente oppure, in alternativa, verrà proposta una maschera di selezione cartelle.

Il comando per la ricerca sarà simile a questo :

"ctags.exe" --recurse=yes --fields=+n --languages=PHP -f "ctags.txt" --langmap=PHP:.php.inc "sources\"

dove 

--languages = linguaggio
ctags.txt = file di output
--lanmap = file da analizzare
sources\ = cartella base dalla quale iniziare la ricerca
