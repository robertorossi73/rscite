[h4]Come riaprire un file utilizzato in precedenza?[/h4]

La prima cosa da sapere riguarda il fatto che RSciTE ha memoria dell'ultima sessione di lavoro. Ogni qual volta il programma viene chiuso, vengono memorizzati i file in uso e riaperti al successivo avviamento di RSciTE. In questo modo è possibile terminare il software senza dover pensare a cosa è aperto.

[h4]Le sessioni e i file aperti[/h4]

[TODO : immagine con evidenziate voci di sessione nel menu file , allineato a sinistra]Infine, una menzione va fatta al salvataggio e ripristino delle sessioni.

Una sessione consente di salvare lo stato dei file correnti, per poi riaprirli in seguito. Supponiamo, ad esempio, da avere una decina di file aperti, usando la voce "Salva Sessione" presente nel menu "File" salviamo i dati della sessione corrente, in un secondo momento quando sarà necessario riaprire gli stessi file, sarà sufficiente ricaricare quanto salvato utilizzando la voce "Carica Sessione", sempre nel menu "File".
